# The Findlay Card Show Website

This is a completed static one-page website for The Findlay Card Show.

## Open the Site

Open `index.html` in a browser.

## Update Event Details

Most editable event details are in `script.js` inside the `SITE_CONFIG` object:

- `showTime`
- `admissionPrice`
- `tablePrice`
- `tablePriceNumeric`
- `availableTables`
- `vendorSetupTime`
- `contactEmail`
- `facebookEventLink`
- `vendorFormLink`
- `refundPolicy`

Replace placeholder values such as `[Show Time]`, `[Vendor Table Price]`, and `[Contact Email]` when final details are available.

## Vendor Registration

Vendor registration is handled through this Google Form:

https://forms.gle/QwTPR3o1Ax1UqKz57

Update `vendorFormLink` in `script.js` and the button link in `index.html` if the registration form changes.
